JaxAHT is a JAX-based benchmark designed to accelerate research across the full lifecycle of **Ad Hoc Teamwork (AHT)** — the problem of training agents that can cooperate with previously unseen teammates. The library provides environments, a suite of AHT and MARL algorithms, and pre-built evaluation tools, all optimized with JAX's JIT compilation for fast iteration.

## What is Ad Hoc Teamwork?

Ad Hoc Teamwork is the challenge of building agents that can cooperate effectively with unknown, potentially heterogeneous partners at test time — without prior coordination or joint training. This differs from standard MARL, where all agents are trained together from scratch.

JaxAHT structures the AHT research lifecycle into four interconnected algorithm categories:

## Key Features

- **JAX-accelerated**: Full JIT compilation of training loops for maximum throughput on GPU/TPU
- **Unified interface**: A shared agent and population interface lets agents trained by one algorithm plug into another — a common AHT workflow
- **Single-file algorithms**: Minimally modularized implementations make it easy to understand and modify any algorithm
- **Heldout evaluation**: Pre-trained evaluation teammates for LBF and Overcooked with downloadable benchmark data
- **Hydra configuration**: Manage algorithm and task configurations from the command line without touching code
- **W&B integration**: Automatic experiment tracking with Weights & Biases

## Environments

JaxAHT supports two cooperative multi-agent environments out of the box:

| Environment | Source | Description |
|-------------|--------|-------------|
| **Level-Based Foraging (LBF)** | Jumanji | Cooperative foraging with 7×7 and 12×12 grid variants |
| **Overcooked-v1** | JaxMARL | Cooperative cooking with 5 layout variants |

<Note>
Both environments include pre-built heldout evaluation sets. Download them with `python download_eval_data.py` after installation.
</Note>
